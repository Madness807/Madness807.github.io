Libft

Recode des fonctions standard C pour une meilleure compréhension

Voir sur GitHub Installer

Description

Libft est un projet fondamental de l'école 42 qui consiste à recoder un ensemble de fonctions de la bibliothèque standard C. Ce projet permet de :

  • Comprendre en profondeur le fonctionnement des fonctions standard
  • Améliorer ses compétences en programmation C
  • Développer une bibliothèque personnelle réutilisable
  • Apprendre les bonnes pratiques de programmation

Technologies Utilisées

Fonctionnalités

Fonctions de base

  • ft_memset
  • ft_bzero
  • ft_memcpy
  • ft_memccpy
  • ft_memmove
  • ft_memchr
  • ft_memcmp

Fonctions de chaînes

  • ft_strlen
  • ft_strdup
  • ft_strcpy
  • ft_strncpy
  • ft_strcat
  • ft_strncat
  • ft_strlcat

Fonctions de caractères

  • ft_isalpha
  • ft_isdigit
  • ft_isalnum
  • ft_isascii
  • ft_isprint
  • ft_toupper
  • ft_tolower

Installation

git clone https://github.com/Madness807/42_libft.git
cd 42_libft
make

Utilisation

Pour utiliser libft dans votre projet :

  1. Copiez le fichier libft.h et libft.a dans votre projet
  2. Incluez libft.h dans vos fichiers source
  3. Liez votre projet avec libft.a lors de la compilation
#include "libft.h"

int main(void)
{
    char str[] = "Hello, World!";
    ft_putstr(str);
    return (0);
}
