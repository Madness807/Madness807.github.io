so_long

Jeu 2D avec minilibx

Voir sur GitHub Installer

Description

so_long est un petit jeu 2D développé avec la bibliothèque graphique minilibx. Le joueur doit collecter tous les objets et atteindre la sortie en évitant les obstacles.

Technologies Utilisées

Fonctionnalités

Graphismes 2D

Rendu graphique avec la bibliothèque minilibx

Gestion des Collisions

Détection des collisions avec les murs et les objets

Système de Score

Comptage des objets collectés et des mouvements

Installation

git clone https://github.com/Madness807/so_long.git

cd so_long

make

Utilisation

Pour lancer le jeu :

./so_long maps/map1.ber

Contrôles :

  • Flèches directionnelles pour se déplacer
  • ESC pour quitter
