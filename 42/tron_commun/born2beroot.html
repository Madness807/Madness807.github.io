Born2beroot

Configuration d'un serveur Debian avec des règles de sécurité strictes

Voir sur GitHub Installer

Description

Born2beroot est un projet qui consiste à configurer un serveur Debian avec des règles de sécurité strictes. Le projet vise à mettre en place un environnement serveur sécurisé et optimisé, en suivant les meilleures pratiques de sécurité.

Technologies Utilisées

Fonctionnalités

Configuration du Pare-feu

Mise en place d'un pare-feu strict avec des règles spécifiques pour les services autorisés.

Gestion des Utilisateurs

Configuration des utilisateurs avec des politiques de mot de passe strictes et des groupes spécifiques.

Monitoring Système

Mise en place d'un système de monitoring pour suivre l'utilisation des ressources.

Installation

1. Créer une machine virtuelle avec VirtualBox

2. Installer Debian 11

3. Configurer le système selon les spécifications

4. Vérifier la conformité avec le script de vérification

Utilisation

Le serveur est configuré pour être utilisé comme un environnement de développement sécurisé. Les utilisateurs peuvent se connecter via SSH en utilisant leurs identifiants configurés.
